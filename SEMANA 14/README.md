# Árbol Binario de Búsqueda (BST) en C#

## Desarrollador
Nixon Dario Corte Salinas

## Descripción
Este proyecto consiste en el desarrollo de una aplicación en consola en C# que permite implementar y gestionar un Árbol Binario de Búsqueda (BST). El programa permite al usuario interactuar con el árbol mediante un menú, realizando operaciones fundamentales como inserción, búsqueda, eliminación y recorridos.

## Objetivos
- Implementar la estructura de datos de un Árbol Binario de Búsqueda.
- Aplicar conceptos de programación orientada a objetos en C#.
- Permitir la interacción del usuario mediante un menú en consola.
- Gestionar y manipular datos de forma eficiente.
  
## Estructura del Proyecto
- **Clase Nodo**: Representa cada elemento del árbol.
  - **Valor**: dato almacenado en el nodo.
  - **Izquierdo**: referencia al hijo izquierdo.
  - **Derecho**: referencia al hijo derecho.
  
- **Clase BST**: Contiene la lógica del árbol binario de búsqueda.
  - Inserción de nodos
  - Búsqueda de valores
  - Eliminación de nodos
  - Recorridos del árbol
  - Cálculo de altura
  - Obtención de valores mínimo y máximo
  - Limpieza del árbol

- **Clase Program**: Contiene el método Main y el menú interactivo que permite al usuario ejecutar las operaciones del BST.

## Funcionalidades
El programa permite realizar las siguientes acciones:
1. Insertar valores en el árbol
2. Buscar un valor
3. Eliminar un valor (considerando los 3 casos posibles)
4. Mostrar recorrido Inorden
5. Mostrar recorrido Preorden
6. Mostrar recorrido Postorden
7. Mostrar el valor mínimo
8. Mostrar el valor máximo
9. Mostrar la altura del árbol
10. Limpiar completamente el árbol

## Tipos de Recorridos
- **Inorden**: Muestra los valores en orden ascendente.
- **Preorden**: Visita primero la raíz, luego izquierda y derecha.
- **Postorden**: Visita izquierda, derecha y luego la raíz.

## Ejecución
1. Abrir el proyecto en Visual Studio o cualquier compilador de C#.
2. Compilar el programa.
3. Ejecutar en consola.
4. Utilizar el menú para interactuar con el árbol.

## Requisitos
- .NET Framework o .NET Core
- Entorno de desarrollo (Visual Studio recomendado)

## Conclusión
Este proyecto permite comprender cómo funciona un Árbol Binario de Búsqueda y cómo se pueden aplicar estructuras de datos para organizar y gestionar información de manera eficiente en programación.
